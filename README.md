# Super Clipboard - 超级剪贴板

## 项目概述
Super Clipboard 是一款功能强大的跨平台剪贴板管理工具，能够自动记录和管理您的所有复制内容。无论是文本、图片还是文件，都能轻松保存和快速找回，大幅提升工作效率。

## 支持平台
- **Windows**：完整功能支持，包括系统托盘、全局热键、开机自启动
- **macOS**：原生体验优化，支持 Dock 交互、窗口透明效果、系统通知
- **Linux**：核心功能完整支持，窗口拖拽调整大小

## 核心功能

### 📋 智能历史记录
- 自动捕获所有复制内容，无需手动操作
- 支持文本、图片、文件等多种内容类型
- 智能去重，避免重复记录相同内容
- 永久保存或自定义保留策略（按天数/数量）

### ⭐ 收藏管理
- 一键收藏重要内容，方便快速访问
- 收藏夹独立筛选，重要信息不丢失
- 支持批量管理收藏项

### 🔍 强大搜索
- 实时关键词搜索，毫秒级响应
- 按内容类型精确筛选（文本/图片/文件）
- 组合搜索条件，快速定位目标内容
- 搜索框位置可自定义

### ⌨️ 高效快捷键
- 全局热键一键唤起剪贴板窗口
- 数字键快速粘贴常用内容
- 方向键浏览历史记录
- 完全可自定义的快捷键配置

### 🎯 多选与批量操作
- Ctrl+Shift 组合键启用多选模式
- 长按方向键连续选择多个项目
- 批量删除无用记录
- 合并多个文本内容为一条

### 🖼️ 卡片式界面
- 现代化的卡片设计，直观美观
- 平滑滚动加载，流畅体验
- 键盘导航支持，无需鼠标即可操作
- 详情弹窗预览，查看完整内容

### 📌 窗口管理
- 窗口置顶功能，随时保持可见
- 系统托盘图标，后台运行不干扰
- 优雅的显示/隐藏动画效果
- 开机自动启动，随时待命

### ⚙️ 个性化设置
- **主题切换**：亮色模式、暗色模式、跟随系统
- **多语言支持**：简体中文、英文等多语言界面
- **行为定制**：
  - 自动收藏新复制内容
  - 自动排序最新内容
  - 删除前确认提示
  - 启动时自动聚焦搜索框
- **历史清理**：灵活配置保留天数和最大记录数

### 🔒 隐私安全
- 所有数据本地存储，不上传云端
- 加密存储敏感配置
- 软删除机制，防止误删恢复

### 💡 贴心细节
- 首次使用自动引导
- 智能识别内容类型并分类
- 复制即保存，零学习成本
- 低资源占用，后台运行无感知

## 适用场景
- **开发者**：快速查找和复用代码片段
- **设计师**：管理截图和素材图片
- **文案工作者**：保存多版本文案草稿
- **日常办公**：追溯历史复制内容，提升效率

## 特色优势
✅ **完全离线**：无需联网，保护隐私  
✅ **跨平台同步体验**：各平台功能一致  
✅ **开箱即用**：安装即可使用，无需复杂配置  
✅ **轻量高效**：启动迅速，内存占用低
